# 👻 DeskGhost

내 맥북 화면을 훔쳐보는 유령 AI. 화면 뒤에 숨어 살면서 주기적으로 스크린샷을 훔쳐보다가,
딴짓하면 화면 구석에 슥 나타나 잔소리를 합니다.

- **잔소리 빌런**: 코딩하다 유튜브/SNS로 새면 → "지금 숏폼 볼 때인가요? 저번에 짜둔 코드 버그나 고치시죠."
- **한숨 유령**: 화면에 에러 로그가 보이면 → 땀 흘리며 깊은 한숨
- **기지개 이스터에그**: 3시간 연속 코딩하면 → 유령이 쭈욱 기지개를 켜며 휴식 권유 (API 호출 없이 로컬 판정)

## 실행

```sh
npm install
export OPENAI_API_KEY=sk-...
npm start
```

메뉴바에 👻 아이콘이 뜨면 성공. 유령 창은 우하단에 투명하게 떠 있고 클릭이 통과하므로
작업을 방해하지 않습니다. 종료/일시정지는 메뉴바 👻 메뉴에서.

## macOS 권한 (첫 실행 시 요청됨)

| 권한 | 용도 | 설정 위치 |
|---|---|---|
| 화면 기록 | 스크린샷 (`screencapture`) | 시스템 설정 → 개인정보 보호 및 보안 → 화면 기록 |
| 자동화 (System Events) | 활성 앱 감지 (`osascript`) | 시스템 설정 → 개인정보 보호 및 보안 → 자동화 |

권한을 허용한 뒤에는 앱을 한 번 재시작하세요.

## 비용

유령이 화면을 "보는" 순간에만 OpenAI API를 호출합니다:

- 15초마다 도는 루프는 활성 앱 이름만 보는 **무료 체크**
- 실제 호출은 **앱이 바뀌었을 때** 또는 **최소 60초 간격**으로만
- 스크린샷은 1280px로 다운스케일해 이미지 토큰 절약

주기와 모델은 [src/config.js](src/config.js)에서 조절할 수 있습니다.
`minCallIntervalMs`를 늘리는 것이 가장 확실한 비용 절감 방법입니다.

## 구조

```
main.js           Electron 메인 — 투명 창, 트레이, 감시 루프(비용 게이트)
preload.js        IPC 브릿지
src/config.js     설정 (주기, 모델, 코딩 앱 목록 등)
src/capture.js    스크린샷 + 활성 앱 감지 (macOS 네이티브 명령, 의존성 0)
src/brain.js      OpenAI 호출 — 페르소나 프롬프트 + structured output
renderer/         유령 캐릭터 (SVG + CSS 애니메이션, 기분별 표정)
```

## 프라이버시

스크린샷은 판정 직후 임시 파일 하나(`/tmp`)에만 남고 어디에도 저장/전송되지 않습니다
(OpenAI API 호출 제외). 페르소나 프롬프트에 화면 속 민감 정보(비밀번호 등)를 언급하지
않도록 규칙이 들어 있습니다.
